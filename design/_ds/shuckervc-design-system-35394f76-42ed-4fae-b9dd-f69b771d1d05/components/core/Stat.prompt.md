**Stat** — a headline metric set in Alice with a sans caption; for fund facts and portfolio KPIs.

```jsx
<Stat value="$8M" label="Bay Area seed fund" />
<Stat value="3.08" label="DPI on prior investments" tone="gold" />
<Stat value="≤ $500K" label="Check size" align="center" />
```

Tones: `ink` · `gold` · `inverse` (dark sections).
